## Deployment Steps

### 1. Sign Up / Log In to Render
- Go to: https://dashboard.render.com/
- Sign up with GitHub (easiest)
- Or create account with email

### 2. Create New Web Service
1. Click **"New +"** button (top right)
2. Select **"Web Service"**

### 3. Connect GitHub Repository
1. Click **"Connect account"** (if first time)
2. Authorize Render to access GitHub
3. Search for: `collaborative-code-editor`
4. Click **"Connect"** on your repository

### 4. Configure Service

Fill in these exact values:

**Name:**
```
collaborative-code-editor
```

**Region:**
```
Oregon (US West) - or closest to you
```

**Branch:**
```
main
```

**Root Directory:**
```
(leave empty)
```

**Runtime:**
```
Python 3
```

**Build Command:**
```
pip install -r requirements.txt
```

**Start Command:**
```
gunicorn --worker-class eventlet -w 1 --bind 0.0.0.0:$PORT code_editor_app:app
```

**Instance Type:**
```
Free
```

### 5. Add Environment Variable

Click **"Advanced"** → **"Add Environment Variable"**

- **Key:** `GROQ_API_KEY`
- **Value:** (paste your Groq API key here)

### 6. Deploy!

1. Click **"Create Web Service"**
2. Wait 3-5 minutes for build to complete
3. You'll see build logs in real-time

## Troubleshooting

**Build fails?**
- Check logs in Render dashboard
- Verify requirements.txt is present
- Check Python version compatibility

**App not starting?**
- Verify start command is correct
- Check GROQ_API_KEY is set
- View logs for error messages

**WebSocket not working?**
- Render supports WebSocket on all plans
- Check browser console for errors
- Verify flask-socketio is installed
